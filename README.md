# Vesak WebXR AR Demo

A simple WebXR AR scene showing Vesak models in a fixed location.

## What is included
- Stone Buddha statue
- Buddha statue
- Lotus flower models
- Lotus lantern
- Vesak lantern
- Sri Lankan stupa

## How to use
1. Open `index.html` in a WebXR-compatible mobile browser (Chrome/Edge on Android, Safari on iOS with WebXR support).
2. Tap the AR button.
3. Point your device at a flat surface.
4. Tap the screen to place the fixed Vesak scene.

## Notes
- Models are loaded directly from GitHub raw URLs.
- The scene is fixed after placement and remains in the same location while the AR session is active.
